### Descripción

Marketplace es una aplicación móvil orientada a la compra y venta de productos al por mayor y menor.

La plataforma permitirá conectar compradores y vendedores en un entorno seguro, organizado y confiable.

### Tecnologías

- Android Studio
- Kotlin
- Spring Boot
- MySQL
- Railway
- GitHub
